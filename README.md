Overview:

This is a simple number guessing game written in Python.
The computer generates a random number between 1 and 100, and the player must try to guess this number in the fewest possible attempts.
The game provides the player with hints after each guess (greater than or less than the correct number).

Features:

• Generates a random number between 1 and 100.
• Hints to help the player narrow down the guess.
• Tracks the number of attempts.
• Simple and easy-to-use command-line interface.
